# FoodTruck

FoodTruck is a group project developed as part of my web development course.
The project was completed by a team of three students.

## Project Overview
The main goal of this project was to learn how to work collaboratively using GitHub
and Docker while developing a web application as a team.

## My Role
- Lead developer in the team
- Coordinated tasks and project structure
- Contributed to frontend development
- Worked with Docker setup and usage
- Collaborated using GitHub (branches, commits, merges)

## Technologies Used
- HTML
- CSS
- Docker
- GitHub (team collaboration)

## What I Learned
- Working in a team environment
- Using GitHub for collaboration
- Understanding Docker basics
- Organizing and leading a small development team

## Status
Completed as a group project for the course.
